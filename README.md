📝 Todo Application

This is a simple Todo list web application built using:

🌿 Spring Boot
🧾 Thymeleaf (for HTML templates)
🐬 MySQL Database

💡 Features

Add new tasks
Mark tasks as completed or not completed (Toggle)
Delete tasks
View all tasks on one page
